# CRIS VŨ — Website catalog áo bóng đá replica

Các trang:
- index.html — Trang chủ
- catalog.html — Sản phẩm (tìm kiếm + lọc theo đội + bộ lọc)
- about.html — Giới thiệu shop
- data/products.json — Danh sách sản phẩm
- images/ — Ảnh thật ({id}-front.jpg, {id}-back.jpg, {id}-detail.jpg)
- assets/ — Giao diện & mã nguồn

Xem thử trên máy: mở thư mục rồi chạy  python3 -m http.server 8000  → vào http://localhost:8000
