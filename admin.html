Đăng nhập quản trị

Dán mã token GitHub của bạn vào đây để quản lý sản phẩm. Token chỉ được giữ trong phiên làm việc này, đóng tab là tự xoá.

Chưa có token? Xem cách lấy
  1. Vào github.com → ảnh đại diện góc phải → Settings
  2. Kéo xuống cuối menu trái → Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Đặt tên bất kỳ; mục Repository access chọn Only select repositories → chọn repo CrisVu
  5. Mục Permissions → Repository permissions → Contents chọn Read and write
  6. Bấm Generate token, sao chép mã (dạng github_pat_...) rồi dán vào ô trên

Giữ token bí mật, không gửi cho ai. Nếu lỡ lộ, vào lại trang token bấm Revoke để huỷ.
